# 🎓 GitHub for Students: Maximizing Your Developer Journey

> *This guide is based directly on official GitHub Education documentation.*

Whether you're starting your first CS course, building a portfolio, or collaborating on a group project, GitHub provides a suite of tools tailored specifically for students.

---

## 🎒 1. The GitHub Student Developer Pack

The **GitHub Student Developer Pack** is your ultimate toolkit. It provides students with free access to the best developer tools in one place so you can learn by doing.

### How to Apply
1. **Visit:** Go to [GitHub Education](https://education.github.com/pack).
2. **Requirements:** You must be currently enrolled in a degree or diploma-granting course of study.
3. **Verification:** Use your school-issued email address, upload a school ID, or provide other proof of enrollment.
4. **Renewal:** The pack is valid for as long as you are a student (you must re-verify periodically).

### Key Benefits Included
- **GitHub Pro:** Free while you are a student (includes advanced code review tools, GitHub Pages, and GitHub Copilot access).
- **Cloud Hosting & Domains:** Free credits from platforms like DigitalOcean, Azure, and AWS, plus free domains from Namecheap or .ME to host your projects.
- **Developer Tools:** Free access to robust tools like JetBrains IDEs, GitKraken, Termius, and Bootstrap Studio.
- **Design & Management:** Access to Canva Pro, Typeform, and icon sets.

## 🏫 2. GitHub Classroom

If your teacher uses **GitHub Classroom**, you'll be interacting with it frequently. It automates repository creation and access control.

### Student Workflow in Classroom
1. **Accept the Assignment:** Click the invite link provided by your instructor.
2. **Access Your Repo:** GitHub automatically generates a new repository (often containing starter code) just for you or your team.
3. **Work locally:**
   ```bash
   git clone <your-assignment-repo-url>
   cd <your-assignment-repo>
   ```
4. **Push your changes:** Work on your assignment, commit, and push back to GitHub.
5. **Autograding:** In many cases, your instructor will set up automated tests via GitHub Actions. You can check the "Actions" tab in your repository to see if your code passes the tests instantly!

## 🤝 3. Building Your Student Portfolio

Your GitHub profile is often your most powerful resume.

- **Pin Your Best Work:** Pin up to 6 repositories on your profile to highlight class projects or personal apps.
- **Create a Profile README:** Use a special `README.md` at `github.com/<your-username>/<your-username>` to introduce yourself, list your tech stack, and describe what you're currently learning.
- **Use GitHub Pages:** Turn your repository into a live website to showcase your projects (great for frontend/web dev portfolios). Look into `gh-pages` branch or deploying via Actions!

---

## 🤖 4. GitHub Copilot for Students

**GitHub Copilot** acts as an AI pair programmer and is completely **FREE** for verified students.

- **Setup:** Once verified as a student, install the Copilot extension in your IDE (VS Code, Visual Studio, JetBrains, etc.).
- **Usage:** It suggests whole lines or entire functions as you type. Use it to understand complex logic, but **do not use it to blindly solve assignments**—use it as a learning tool to see how professional code is structured.
- **Copilot Chat:** Chat directly inside your editor to ask questions like, "Explain this sorting algorithm" or "Why is this code throwing a NullReferenceException?"

## 📚 5. GitHub Global Campus

Consider joining **GitHub Global Campus**, a portal for students to manage their tools, connect with campus experts, and discover events.

- **Campus Experts:** A program for student leaders who build tech communities on their campus. If you love helping others learn, consider applying!
- **Hackathons:** Discover student-friendly hackathons supported by MLH (Major League Hacking) directly from your dashboard.

> **Pro Tip:** Always keep your student email attached to your GitHub account to maintain these benefits, but set a personal email as your primary address for account recovery after graduation!

## 🎓 6. What Happens When You Graduate?

Once you graduate and lose access to your university email, your Student Developer Pack access will expire at the end of your current billing cycle or verification period.
- **GitHub Pro:** Reverts to GitHub Free, but you don't lose any repositories.
- **Private Repos:** They stay private! GitHub now offers unlimited free private repositories.
- **Copilot:** You will need to purchase a personal subscription to continue using GitHub Copilot.
